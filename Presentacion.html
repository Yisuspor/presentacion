Composiciones internas:

Son aquellas que parten de un producto cartesiano entre dos tablas.

Composiciones externas:

No se originan de un producto cartesiano sino que se toma un registro de una tabla y se busca registros de otra tabla que coincidan en el atributo indicado.
Composiciones internas:
mysql> SELECT * FROM (nombretabla1) JOIN (nombretabla2);
mysql> SELECT * FROM (nombretabla1) JOIN (nombretabla2) ON ((nombretabla1).id = (nombretabla2).id);
mysql> SELECT * FROM (nombretabla1) NATURAL JOIN (nombretabla2);

Regresar
Composiciones externas
mysql> SELECT * FROM (nombretabla1) LEFT JOIN (nombretabla2) ON ((nombretabla1).id=(nombretabla2).id);
mysql> SELECT * FROM (nombretabla1) RIGHT JOIN (nombretabla2) ON ((nombretabla1).id=(nombretabla2).id);
mysql> SELECT * FROM (nombretabla1) NATURAL RIGHT JOIN (nombretabla2);
mysql> SELECT * FROM (nombretabla1) NATURAL LEFT JOIN (nombretabla2);
